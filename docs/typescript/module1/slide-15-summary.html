Tóm Tắt & Key Takeaways

Variables: const (mặc định), let (biến thay đổi), ❌ var (không dùng).

Naming: camelCase (biến), PascalCase (class/interface), UPPER_SNAKE (config).

Data Types: string, number, boolean, array là 4 kiểu 'xương sống'.

Advanced: Dùng unknown thay cho any để tăng độ an toàn.

Tools: Tận dụng Interface để quản lý Test Data hiệu quả.

TypeScript giúp viết các bài kiểm thử một cách tự tin.
