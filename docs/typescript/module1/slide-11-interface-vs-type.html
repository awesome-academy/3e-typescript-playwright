Interface vs Type Alias

Đặc điểm interface type
Mở rộng Dùng 'extends' Dùng '&' (Intersection)
Declaration Merging ✅ Có thể ❌ Không thể
Union Types ❌ Không hỗ trợ ✅ Hỗ trợ tốt (A | B)
Phù hợp cho Định nghĩa Object, Class Union, Tuple, Complex Type

Best Practice: QA nên dùng interface cho Page Objects và type cho các trạng thái (Status).
