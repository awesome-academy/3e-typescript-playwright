Các kiểu dữ liệu QA thường gặp nhất

String

Dùng cho Selectors, URLs, Text nội dung, XPath.

const loginBtn: string = "#login-button";
const url: string = "https://example.com";

Number

Dùng cho Timeouts, Status Codes, Count phần tử.

const timeout: number = 5000;
const statusCode: number = 200;

Boolean

Dùng làm Flag (isDisplayed, isEnabled, hasError).

const isVisible: boolean = true;
const isEnabled: boolean = false;

Array

Danh sách Users, danh sách Selectors, tập hợp Test Data.

const users: string[] = ["admin", "user1"];
const selectors: string[] = ["#btn", ".link"];
