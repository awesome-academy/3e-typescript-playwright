Sự "lỏng lẻo" của any vs unknown

Đặc điểm any unknown
Type Checking ❌ Tắt hoàn toàn ✅ Luôn kiểm tra
Type Safety ❌ Không an toàn ✅ Rất an toàn
Gán cho kiểu khác ✅ Tự do ❌ Phải check type trước
Best Practice ❌ Tránh lạm dụng ✅ Ưu tiên sử dụng

Lưu ý: Dùng unknown khi nhận dữ liệu từ API không rõ cấu trúc.
